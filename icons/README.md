# Placeholder Icons

The icon files (icon16.png, icon48.png, icon128.png) are required for the Chrome extension.

## Generate Icons

You can create PNG icons from the SVG file:

### Option 1: Use Online Converter

1. Open `icon.svg` in a browser
2. Use an online SVG to PNG converter (e.g., https://cloudconvert.com/svg-to-png)
3. Generate three sizes: 16x16, 48x48, 128x128
4. Save them as `icon16.png`, `icon48.png`, `icon128.png` in this folder

### Option 2: Use Inkscape or GIMP

1. Open `icon.svg` in Inkscape or GIMP
2. Export as PNG with different sizes (16, 48, 128 pixels)
3. Save in this folder

### Option 3: Quick Placeholder

For testing, you can temporarily use any PNG images renamed to the required sizes.

## Temporary Workaround

Until you create the PNG icons, the extension will work but may show a default icon in Chrome.
